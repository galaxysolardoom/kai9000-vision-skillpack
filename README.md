# TeKi-Nano'M Vision Skills 🦋

SKILL.md пакети за Kai 9000 — добавят визуално разпознаване (изображения, видео, обекти, сегментация).

## Инсталация в Kai 9000

Отвори Settings → Tools → Skills → Add Skill и въведи:

| Умение | GitHub reference |
|--------|-----------------|
| 🔍 YOLO Detection | `galaxysolardoom/teki-nanom-vision-skills/yolo-detection` |
| ✂️ SAM 2 Segmentation | `galaxysolardoom/teki-nanom-vision-skills/sam2-segmentation` |
| 🧠 LLaVA Vision | `galaxysolardoom/teki-nanom-vision-skills/llava-vision` |

### Изисквания
- Linux sandbox инсталиран
- ~2-5 GB свободно пространство за моделите (SAM large = 2.5 GB, LLaVA = 16 GB)
- Първото извикване сваля модела автоматично
